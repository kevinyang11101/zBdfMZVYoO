# 学生成绩管理系统 java127

## 项目概述

学生成绩管理系统是一个基于 Java + SpringBoot + Vue 实现的前后端分离项目。该系统包含完整的源码、数据库脚本及相关文档。

## 技术栈

- 后端：Java，SpringBoot
- 前端：Vue
- 运行环境：IDEA，JDK 1.8，MySQL 5.7，Node.js 16，Maven

## 功能模块

### 系统角色

- 学生
- 教师
- 管理员

### 用户模块

- 学生登录
- 教师登录
- 管理员登录

### 管理员功能

- 课程信息管理
- 课程表管理
- 成绩信息管理
- 成绩统计分析
- 用户管理
- 账号管理

- 成绩统计

### 教师功能

- 学生成绩查询
